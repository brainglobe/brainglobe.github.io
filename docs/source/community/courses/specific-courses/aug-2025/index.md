# August 12th-13th 2025 (Neuroinformatics Unit Open Software Week)

## Open Software Week (BrainGlobe track)
[Sainsbury Wellcome Centre, University College London](https://www.sainsburywellcome.org/web/), 12th & 13th August.

## Overview
This course is part of the [Neuroinformatics Unit Open Software Week](https://neuroinformatics.dev/open-software-summer-school/2025/index.html). 
For more details see the [BrainGlobe track page](https://neuroinformatics.dev/open-software-summer-school/2025/brainglobe.html#track-brainglobe).
