# August 24th-28th 2026 (Neuroinformatics Unit Open Software Summer School)

## Open Software Summer School (BrainGlobe track)
[Sainsbury Wellcome Centre, University College London](https://www.sainsburywellcome.org/web/), 24th-28th August.

## Overview
This course is part of the [Neuroinformatics Unit Open Software Summer School](https://neuroinformatics.dev/open-software-summer-school/index.html).
For more details see the [BrainGlobe track page](https://neuroinformatics.dev/open-software-summer-school/2026/brainglobe.html#track-brainglobe-2026).

:::{note}
Application deadline for this course is January 31st 2026. See the 
[OSSS page](https://neuroinformatics.dev/open-software-summer-school/index.html) for more details. 
:::
